# Empty Line Remover

A Python script that removes empty lines from a text file.

## Description
This project reads a text file, skips blank lines, and writes the cleaned content into a new file.

## Features
- File reading
- File writing
- Removes blank lines
- Beginner-friendly implementation

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Place your input file in the project folder
3. Run:

   python remove_empty_lines.py

4. Enter input and output file names

## Example

Input:
input.txt
output.txt

Output:
Empty lines removed successfully.
